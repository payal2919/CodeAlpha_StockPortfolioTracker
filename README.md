# CodeAlpha Stock Portfolio Tracker

## Description
This project is a simple Stock Portfolio Tracker developed using Python as part of the CodeAlpha Python Programming Internship.

## Features
- User enters stock name
- User enters quantity
- Uses a predefined dictionary of stock prices
- Calculates total investment value
- Saves portfolio details to a text file

## Technologies Used
- Python 3

## Files
- main.py
- portfolio.txt

## How to Run
1. Open the project folder.
2. Run the following command:

```bash
python main.py
```

## Learning Concepts
- Dictionaries
- User Input
- Arithmetic Operations
- File Handling
